# VueCast

Monorepo for VueCast tooling. Currently focused on Astro, enabling Vue-style template syntax inside `.astro` files.

## Disclaimer

VueCast is a proof of concept (POC). Editor support (for example, Visual Studio / VS Code syntax tooling) is not available yet.
